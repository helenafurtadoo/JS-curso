
---------OPERADORES DO JS parte 2--------
* relacionais: todo o resultado aqui, vai ser boolean (true | false)
>
<
>=
<=
== --> nao analisa o tipo, e sim o valor. Entao: 5 == "5" TRUE - mesmo um sendo number e outro uma string (NO JS)
    === --> Identidade ex: 5 === "5" FALSE
            5 === 5 TRUE
!= --> Diferente  ex: n1 != n2 --> "n1 é diferente que n2"

* lógicos
!  --> negacao  - true
                - false

&& --> conjuncao (E) - operador binario (retorna falso quando pelo menos UM é false)
true && true --> true
true && false --> false

|| --> dijunçao (OU) - (retorna false quando nao tem NENHUM true)
true || true --> true
true || false --> true

-----EXEMPLOS-----
idade >= 15 && idade <= 17
    "a idade esta entre 15 e 17 incluindo esses dois valores"

estado == "RJ" || estado == "SP"
    "o estado é RJ OU SP?"

salario > 1500 && sexo != "M"
    "é uma mulher que ganha mais que 1500"

* ternário (sempre o ultimo a ser feito)
? : (aparecem na mesma expressao)
possui 3 blocos (porque junta 3 operando)
teste  ?  true  :  false

? --> (se for .... retorna x)
: --> (se nao... retorna y)
exemplo:
var media = 5
media > 7? "APROVADO" : "REPROVADO"

    * como a media nao é maior que 7, entao esta reprovado
