exemplo: 
        <script> // isso é um comentario em js
        var n1 = window.prompt("Digite um numero.") //string
        var n2 = window.prompt("Digite o outro numero") // string

        //somar ambas as variaveis
        var soma = n1 + n2
        var mostrarSoma = window.alert("A soma dos valores é: " + soma)
        <script>
        --- explicação: Aqui, se colocar como n1 = 2, n2 = 2, a soma vai mostrar "22" e nao "4". Pq ambas as variaveis estao como string

        * por isso, aqui é necessario CONVERSÃO, ja que a variavel é criada como string, mas que responder como number para fazer a soma

        - conversao STRING --> NUMBER (pode ser feito direto na variavel, )
            Number.parseInt(n) - quando forem numeros inteiros

            Number.parseFloat(n) - quando forem numeros com virgula (ex: 5.5)

            Number(n) - para que o js interprete de acordo com o que for digitado, e entender se é inteiro ou com virgula, para fazer a operação

        - conversao NUMBER --> STRING
            String(n) - (n --> o que quer ser convertido para string)

            n.toString() - (n --> o que quer converte)
