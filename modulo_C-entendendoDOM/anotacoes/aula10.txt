---------EVENTOS DOM---------
* podem ser configurados direto no HTML(dentro de divs) ou no JS

-- HTML ex:
<div id = 'area' onclick = "clicar()" onmouseenter="entrar()" onmouseout="sair()">

 var a = window.document.getElementById('area') 
 
        function clicar() { //tudo aqui dentro, so sera executado quando chamar o clicar()

            a.innerText = 'Clicou! Agora saia com o mouse'
            a.style.background = 'blue'
        }

-- JS exemplo:
 var a = window.document.getElementById('area')
        a.addEventListener('click', clicar)
    

        function clicar() { //tudo aqui dentro, so sera executado quando chamar o clicar()

            a.innerText = 'Clicou! Agora saia com o mouse'
            a.style.background = 'blue'
        }

alguns eventos de mouse: ('procurar JavaSript DOM events' para documentacao)
- se temos por exemplo uma <div> , os eventos sao tudo aquilo pode acontecer com aquela div

* mouseenter (mouse 'entrou' na div)

* mousemove (movendo o mouse dentro da div)

* mousedown (pressiona com o mouse na div)

* mouseup (o movimento de soltar o botao apertado do mouse)

* click (apertar e soltar rapidamente o mouse)

* mouseout (quando tira o cursor do mouse de dentro da div)
